# Online Resume

## Installation

### Clone the repo

```bash
git clone https://github.com/heykos/resume.git
```

### Install Ruby Gems

```bash
bundle install
```

### Run the site locally

```bash
bundle exec jekyll serve
```

### Navigate to `http://localhost:4000/resume/`
